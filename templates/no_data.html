How to Add Your Data File

1
Prepare Your Excel File
Ensure your Excel file contains monthly sheets with employee attendance data (WFO, WFH, SL, India Holiday, etc.)
2
Create Data Folder
Create a folder named data in your repository root directory
3
Upload Excel File
Place your Excel file (.xlsx) in the data folder
4
Deploy to Azure
Commit and push the changes to deploy the updated application with your data
Repository Structure

Your repository should look like this:

your-repo/
├── app.py
├── requirements.txt
├── templates/
├── data/
│ └── your-wfo-data.xlsx
└── README.md
